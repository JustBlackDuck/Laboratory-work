# Лабораторные работы по дисциплине "Программирование" (C++)

Данный репозиторий содержит лабораторные работы, выполненные на языке **C++**.

## Структура

- **lab1/** — Основы синтаксиса C++, ввод-вывод, переменные
- **lab2/** — Условные операторы и циклы
- **lab3/** — Функции, массивы, строки

> Каждая папка содержит исходный файл `.cpp` и, при необходимости, краткое задание в комментариях.

## Требования

- Компилятор: `g++` (GCC) или `clang++`
- ОС: Windows, Linux или macOS

## Сборка и запуск

Пример для `lab1`:

```bash
cd lab1
g++ -std=c++17 -O2 -Wall -Wextra lab1.cpp -o lab1
./lab1          # Linux/macOS
# или
lab1.exe        # Windows
